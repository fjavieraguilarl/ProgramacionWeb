Introducción

La programación web, parte de las siglas www, que significa World Wide Web o telaraña mundial. Para realizar una página con la programación web se debe tener claro tres conceptos fundamentales los cuales son, el URL (Uniform Resource Locators), es un sistema con el cual localiza un recurso dentro de la red, este recurso puede ser una página web, un servicio o cualquier otra cosa. En resumen el URL no es más que un nombre, que identifica una computadora, dentro de esa computadora un archivo que indica el camino al recurso que solicitas.

El siguiente concepto dentro de la programación web, es el protocolo encargado de llevar la información que contiene una página web por toda la red de internet, como es el HTTP (Hypertext Transfer Protocol). Y por último el lenguaje necesario cuya funcionalidad es la de representar cualquier clase de información que se encuentre almacenada en una página web, este lenguaje es HTML (Hypertext Markup Lenguage). En la programación web, el HTML es el lenguaje que permite codificar o preparar documentos de hipertexto, que viene a ser el lenguaje común para la construcción de una página web.

Con el comienzo de internet y la programación web, se desfasaron los diseños gráficos tradicionales, con lo que se empezaron a diseñar interfaces concretas para este medio, buscando ficheros pequeños para facilitar la carga de los mismos. La programación web se orienta a un diseño muy cargado e interactuando con el usuario.
